台股TradingView Widget範例

範例圖表

功能特色:
• 專業級TradingView圖表介面
• 台股專用符號格式 (TPE:代號)
• 內建技術指標 (成交量、RSI、MACD)
• 響應式設計，支援各種螢幕尺寸
• 詳細的股票資訊面板
• 產業分類和市值資訊
2330 (台積電) 2454 (聯發科) 0050 (台灣50)

API端點

圖表顯示: /chart/taiwan-widget/{symbol}

股票資訊: /api/taiwan-widget/stock-info/{symbol}

符號搜尋: /api/taiwan-widget/symbol-search?query={關鍵字}

⚙ 使用方式

1. 啟動FastAPI服務器: uv run python -m uvicorn src.api.main:app --reload --host 0.0.0.0 --port 8000

2. 訪問圖表: http://localhost:8000/chart/taiwan-widget/2330

3. 或直接開啟上方的範例HTML文件
